# AMD-Mountain-Region-Bioinformatics-Code-Club
The AMD Mountain Region Bioinformatics Code Club is a 30-minute virtual session designed for public health professionals who want to strengthen their bioinformatics skills. Each session is hands-on, using GitHub Codespaces to explore real-world public health data and tools.

## Step-by-Step Documentation
Each session directory contains a dedicated `README.md` file that provides a detailed, step-by-step guide to the curriculum. This structure is designed to support:
* **Live Code-Alongs**: Real-time instruction during the scheduled Zoom meetings.
* **Asynchronous Learning**: Independent completion of the exercises for those unable to attend the live session.
* **Post-Session Review**: A permanent reference for participants to revisit specific commands and technical concepts.

## Accessing the Computational Environment

[![Open in GitHub Codespaces](https://github.com/codespaces/badge.svg)](https://codespaces.new/erinyoung/AMD-Mountain-Region-Bioinformatics-Code-Club)

1. **Navigate to GitHub Codespaces**: https://github.com/codespaces
2. **Create a new codepsace** : Use the following settings:
- Repository: [erinyoung/AMD-Mountain-Region-Bioinformatics-Code-Club](https://github.com/erinyoung/AMD-Mountain-Region-Bioinformatics-Code-Club)
- Branch: main (default)
- Region: US West (or whatever the default is for the user)
- Machine type: 2 core
3.  **Create Codespace**: Click the green button labeled **Create codespace**. 

### Hardware Configuration
The **Free Tier (2-core, 4GB RAM, 32GB Storage)** machine type is the standard selection for these training sessions. This configuration provides sufficient resources for the bioinformatic tasks performed in the club. If a task requires higher specifications, specific instructions will be provided at the beginning of that session.

### Environment Management
* **Auto-Stop**: Codespaces are configured to stop automatically after a period of inactivity to manage usage hours. While work is preserved, the terminal session may require a restart upon returning.
* **Deletion**: It is recommended that Codespaces be deleted after the conclusion of session tasks to stay within GitHub account storage limits.

### DataCamp for Bioinformatics
DataCamp provides structured, interactive courses that assist in learning the basics of the tools used in this club. These modules are tailored to practical applications and allow for skill development in a controlled environment.
